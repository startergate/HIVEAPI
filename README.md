# HIVE

HIVE is movie hive. / HIVE는 영화들이 모여있는 벌집입니다.

## Getting Started / 어떻게 시작하나요?

### Prerequisites / 선행 조건

아래 사항들이 설치가 되어있어야합니다.

```
Node.js 8 or Higher
MongoDB latest
```

### Installing / 설치

Use scripts below to install this project.

```sh
npm install
```

If you have [yarn](https://yarnpkg.com/lang/en/) installed...
```sh
yarn
```
